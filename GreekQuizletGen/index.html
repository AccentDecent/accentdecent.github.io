Greek Quizlet Gen

by CorruptionHades

Tones:


Acute: 1 ώ
Grave: 2 ὼ
Circumflex: 3 ῶ

Smooth: % ὠ
Rough: & ὡ
Iota-Sub: _ ῳ
Diaerisis: ^

eg: a1_ = ᾴ

How to use

- Enter text which will be converted to greek.

- You can use numbers for the tones

- (optionally) you can make it a quizlet set by using this format:

vocab:translation

- use a # at the start to ignore the line
